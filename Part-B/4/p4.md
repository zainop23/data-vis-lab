# Tableau Dashboard Workflow

## Step 1: Load Dataset
1. Open Tableau Desktop
2. Click:
   - Microsoft Excel / Text File
3. Load the dataset
4. In **Data Source** tab:
   - Set `Date` → Date type
   - Set `Revenue`, `Expenses`, `Budget` → Measures
   - Set `Department` → Dimension
5. Open **Sheet 1**

---

## Step 2: Create Calculated Fields

### Profit
- Right-click → Create Calculated Field
- Name: `Profit`

```tableau
SUM([Revenue]) - SUM([Expenses])
```

### Profit Margin
- Right-click → Create Calculated Field
- Name: `Profit Margin`

```tableau
[Profit] / SUM([Revenue])
```

---

## Step 3: Monthly Revenue vs Expenses
1. Drag `Date` → Columns
2. Select:
   - Month
3. Drag:
   - `Revenue` → Rows
   - `Expenses` → Rows
4. Change Marks type:
   - Line Chart
5. Apply:
   - Dual Axis
   - Synchronize Axis
6. Drag:
   - `Measure Names` → Color
7. Rename sheet:
   - `Monthly Revenue vs Expenses`

---

## Step 4: Yearly Revenue vs Expenses
1. Create New Sheet
2. Drag `Date` → Columns
3. Select:
   - Year
4. Drag:
   - `Revenue` → Rows
   - `Expenses` → Rows
5. Apply:
   - Line Chart
   - Dual Axis
   - Synchronize Axis
6. Rename:
   - `Yearly Revenue vs Expenses`

---

## Step 5: Profit Margin by Department
1. Create New Sheet
2. Drag:
   - `Department` → Columns
   - `Profit Margin` → Rows
3. Select:
   - Bar Chart
4. Rename:
   - `Profit Margin by Department`

---

## Step 6: Department Expenditure
1. Create New Sheet
2. Drag:
   - `Department` → Columns
   - `Expenses` → Rows
3. Select:
   - Bar Chart
4. Rename:
   - `Department Expenditure`

---

## Step 7: Budget vs Actual
1. Create New Sheet
2. Drag `Date` → Columns
3. Select:
   - Month
4. Drag:
   - `Revenue` → Rows
   - `Budget` → Rows
5. Apply:
   - Line Chart
   - Dual Axis
   - Synchronize Axis
6. Rename:
   - `Budget vs Actual`

---

## Step 8: Add Filters

### Year Filter
1. Drag `Date` → Filters
2. Select:
   - Years
3. Right-click → Show Filter
4. Set style:
   - Dropdown

### Department Filter
1. Drag `Department` → Filters
2. Click OK
3. Right-click → Show Filter
4. Set style:
   - Dropdown

---

## Step 9: Apply Filters to All Sheets
- Right-click filter →
  `Apply to Worksheets → All Using This Data Source`

---

## Step 10: Create Dashboard
1. Click:
   - Dashboard → New Dashboard
2. Drag all sheets into dashboard:
   - Monthly Revenue vs Expenses
   - Yearly Revenue vs Expenses
   - Profit Margin by Department
   - Department Expenditure
   - Budget vs Actual

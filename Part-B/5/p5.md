
## Step 1: Open Tableau Desktop
1. Open Tableau Desktop
2. On the start screen:
   - Click `Microsoft Excel` for Excel datasets
   - OR click `Text File` for CSV datasets

---

## Step 2: Load Dataset
1. Select the dataset file
2. In the **Data Source** tab, verify:
   - `Date` → Date type
   - `Region` → Dimension
   - `Cases`, `Recoveries`, `Deaths`, `Tests Conducted` → Measures

### If Date is incorrect
- Right-click `Date`
- Select:
  `Change Data Type → Date`

---

## Step 3: Total Cases, Recoveries, and Deaths
1. Open `Sheet 1`
2. Drag:
   - `Measure Names` → Columns
   - `Measure Values` → Rows
3. Keep only:
   - `Total Cases`
   - `Recoveries`
   - `Deaths`
4. Select:
   - Bar Chart
5. Rename sheet:
   - `Total Cases Recoveries and Deaths`

---

## Step 4: Region-wise Case Distribution
1. Create New Sheet
2. Drag:
   - `Region` → Columns
   - `Total Cases` → Rows
3. Select:
   - Bar Chart
4. Sort in descending order
5. Rename:
   - `Region-wise Case Distribution`

---

## Step 5: Daily Trend Analysis
1. Create New Sheet
2. Drag:
   - `Date` → Columns
3. Select:
   - Day (Continuous)
4. Drag:
   - `Total Cases` → Rows
5. Select:
   - Line Chart
6. Rename:
   - `Daily Trend Analysis`

---

## Step 6: Cumulative Trend Analysis
1. Create New Sheet
2. Drag:
   - `Date` → Columns
   - `Total Cases` → Rows
3. Right-click `Total Cases`
4. Select:
   `Quick Table Calculation → Running Total`
5. Select:
   - Line Chart
6. Rename:
   - `Cumulative Trend Analysis`

---

## Step 7: Create Positivity Rate
1. Right-click in Data Pane
2. Select:
   `Create Calculated Field`
3. Name:
   `Positivity Rate`

```tableau
([Total Cases] / [Tests Conducted]) * 100
```

4. Click OK

---

## Step 8: Format Positivity Rate
1. Right-click `Positivity Rate`
2. Select:
   `Default Properties → Number Format`
3. Choose:
   - Percentage
4. Set decimal places:
   - 2
5. Click OK

---

## Step 9: Testing and Positivity Rate Analysis
1. Create New Sheet
2. Drag:
   - `Date` → Columns
   - `Tests Conducted` → Rows
   - `Positivity Rate` → Rows
3. Select:
   - Line Chart
4. Apply:
   - Dual Axis

### Important
- Do NOT use Synchronize Axis because:
  - Tests Conducted uses large values
  - Positivity Rate uses percentage values

5. Rename:
   - `Testing and Positivity Rate`

---

## Step 10: Add Filters

### Region Filter
1. Drag `Region` → Filters
2. Click OK
3. Right-click `Region`
4. Select:
   `Show Filter`
5. Choose:
   - Dropdown

### Date Filter
1. Drag `Date` → Filters
2. Select:
   - Range of Dates
3. Right-click `Date`
4. Select:
   `Show Filter`

---

## Step 11: Apply Filters to All Worksheets
- Right-click filter
- Select:

```text
Apply to Worksheets → All Using This Data Source
```

---

## Step 12: Create Dashboard
1. Click:
   `Dashboard → New Dashboard`
2. Add all worksheets:
   - Total Cases Recoveries and Deaths
   - Region-wise Case Distribution
   - Daily Trend Analysis
   - Cumulative Trend Analysis
   - Testing and Positivity Rate
3. Place filters on top or side

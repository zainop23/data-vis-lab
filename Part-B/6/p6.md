# Tableau Hospital Management Dashboard Workflow

## Step 1: Open Tableau Desktop
1. Open Tableau Desktop
2. On the start screen:
   - Click `Microsoft Excel` for Excel datasets
   - OR click `Text File` for CSV datasets

---

## Step 2: Load Dataset
1. Select the dataset file
2. In the **Data Source** tab, verify:
   - `Admission Date` and `Discharge Date` Ôćĺ Date type
   - `Department` and `Doctor Name` Ôćĺ Dimensions
   - `Length of Stay` Ôćĺ Measure

### If dates are incorrect
- Right-click field
- Select:
  `Change Data Type Ôćĺ Date`

---

## Step 3: Patient Status Analysis
1. Open `Sheet 1`
2. Drag:
   - `Patient Status` Ôćĺ Columns
   - `Hospital_Patient_Data (Count)` Ôćĺ Rows
3. Select:
   - Bar Chart
4. Rename:
   - `Patient Status Analysis`

---

## Step 4: Department-wise Patient Distribution
1. Create New Sheet
2. Drag:
   - `Department` Ôćĺ Columns
   - `Hospital_Patient_Data (Count)` Ôćĺ Rows
3. Select:
   - Bar Chart
4. Sort in descending order
5. Rename:
   - `Department-wise Patient Distribution`

---

## Step 5: Doctor-wise Patient Load
1. Create New Sheet
2. Drag:
   - `Doctor Name` Ôćĺ Columns
   - `Hospital_Patient_Data (Count)` Ôćĺ Rows
3. Select:
   - Bar Chart
4. Sort by patient count
5. Rename:
   - `Doctor-wise Patient Load`

---

## Step 6: Average Hospital Stay
1. Create New Sheet
2. Drag:
   - `Department` Ôćĺ Columns
   - `Length of Stay` Ôćĺ Rows
3. Change aggregation:
   - Right-click `Length of Stay`
   - Select:
     `Measure Ôćĺ Average`
4. Select:
   - Bar Chart
5. Rename:
   - `Average Hospital Stay`

---

## Step 7: Create Length of Stay (Optional)
1. Right-click Data Pane
2. Select:
   `Create Calculated Field`
3. Name:
   `Length of Stay`

```tableau
DATEDIFF('day', [Admission Date], [Discharge Date])
```

4. Click OK

---

## Step 8: Add Filters

### Department Filter
1. Drag `Department` Ôćĺ Filters
2. Click OK
3. Right-click `Department`
4. Select:
   `Show Filter`
5. Choose:
   - Dropdown

### Doctor Filter
1. Drag `Doctor Name` Ôćĺ Filters
2. Click OK
3. Right-click `Doctor Name`
4. Select:
   `Show Filter`

### Date Range Filter
1. Drag `Admission Date` Ôćĺ Filters
2. Select:
   - Range of Dates
3. Right-click `Admission Date`
4. Select:
   `Show Filter`

---

## Step 9: Apply Filters to All Worksheets
- Right-click filter
- Select:

```text
Apply to Worksheets Ôćĺ All Using This Data Source
```

---

## Step 10: Create Dashboard
1. Click:
   `Dashboard Ôćĺ New Dashboard`
2. Add all worksheets:
   - Patient Status Analysis
   - Department-wise Patient Distribution
   - Doctor-wise Patient Load
   - Average Hospital Stay
3. Place filters on top or side

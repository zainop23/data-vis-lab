Part A: Data Visualization with Python
1. Write a program to perform time series visualization using Pandas and Matplotlib,
showing trends, seasonal patterns, and moving averages.
Given the "Airline Passengers" dataset (1949–1960), write a Python program to:
- Pre-process the data by converting date strings into a DatetimeIndex.
- Calculate a 12-month Rolling Mean to smooth out noise and visualize the growth trend.
- Generate a Seasonal Boxplot to identify recurring monthly fluctuations (Seasonality).

2. Write a program to create pair plots and regression plots using Seaborn, to study variate relationships and trends.

3. Write a program to create interactive visualizations using Plotly, including interactive line, bar, and scatter plots with hover information.

4. Write a program to design an interactive dashboard using Plotly, incorporating multiple charts, filters, and dropdown controls.

5. Write a program to visualize geographical data, using map-based plots to represent location-based information.

6. Write a program to visualize machine learning model outputs, including confusion matrices, ROC curves, and feature importance plots.

Part B : Power BI and Tableau

1. A retail company wants to analyse its sales performance across different
regions, product categories, and products. Using Power BI Desktop,
complete the following tasks and combine them into a single interactive
dashboard:
a. Visualize total sales, total profit, and sales growth.
b. Create charts for sales by region, category, and sub-category
c. Plot monthly and yearly sales trends.
d. Identify the top 10 best-selling products.
e. Implement interactive filters for year, region, and category.

2. An e-commerce company wants insights into customer behavior and
sales. Using Power BI Desktop, complete the following tasks and combine
them into a single interactive dashboard:
a. Visualize total orders, revenue, and average order value.
b. Perform customer segmentation by location and age group.
c. Analyse product category performance.
d. Analyse order status (delivered, returned, cancelled).
e. Implement filters for time period, location, and category.

3. A logistics firm wants to monitor transportation efficiency. Using Power
BI Desktop, complete the following tasks and combine them into a single
interactive dashboard:
a. Show on-time versus delayed deliveries.
b. Visualize route-wise delivery performance.
c. Analyse vehicle utilization and Plot fuel cost trends.
d. Apply filters for route, vehicle type, and date.

4. A finance department wants to analyse revenue, expenses, and profit.
Using Tableau Desktop, complete the following tasks and combine them
into a single interactive dashboard:
a. Visualize monthly and yearly revenue versus expenses.
b. Analyze profit margins and Show department-wise expenditure.
c. Compare budget versus actual values.
d. Implement filters for year and department.

5. A health authority wants to track disease spread and recovery trends.
Using Tableau Desktop, complete the following tasks and combine them
into a single interactive dashboard:
a. Visualize total cases, recoveries, and deaths.
b. Display region-wise case distribution.
c. Plot daily and cumulative trend analysis.
d. Calculate testing and positivity rates.
e. Apply filters for region and date.

6. A hospital wants to visualize patient and operational data to improve
decision-making. Using Tableau Desktop, complete the following tasks and
combine them into a single interactive dashboard:
a. Show the number of patients admitted, discharged, and under treatment.
b. Visualize department-wise patient distribution.
c. Analyze doctor-wise patient load and Calculate the average length of
hospital stay.
d. Apply filters for department, doctor, and date range.
